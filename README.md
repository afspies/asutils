# asutils

Personal CLI utilities.

## Install

```bash
pip install asutils
```

## Commands

| Command | Description |
|---------|-------------|
| `as-repo init [name]` | Scaffold new Python project |
| `as-git sync` | Quick add, commit, push |
| `as-publish release` | Publish to PyPI |

## Dev Install

```bash
git clone https://github.com/afspies/asutils
cd asutils
uv pip install -e ".[dev]"
```
